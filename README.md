Projekt został podzielony na 5 klas.
Program.cs - UI w konsoli 
Sprzet.cs - klasa abstrakcyjna Sprzet i dziedziczące po niej 3 klasy sprzętów
Uzytkownik.cs - klasa abstrakcyjna Uzytkownik i dziedziczące po niej 2 klasy uzytkownikow
Wypozyczalnia.cs - klasa odpowiedzialna za całą funkcjonalność kodu, to w niej stworzone są metody używane w UI
Wypozyczenie.cs - klasa odpowiedzialna za przetrzymywanie danych wypozyczen oraz obliczanie kary

Projekt jest symulatorem wypozyczalni sprzetu
Po odpaleniu pyta użytkownika co chce zrobic z listy, po wybraniu pokazuje instrukcje co robic.
